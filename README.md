# Android Studio Projects - MM Internship

## Overview
Collection of Android apps developed during MM Internship. All projects are clean (no build artifacts, caches, or IDE files committed).

## Projects

| Project | Description |
|---------|-------------|
| Affirmations | Affirmations display app with data source & RecyclerView |
| ArtGalleryApp | Art gallery image viewer |
| BusinessCardApp | Business card UI app |
| DiceRoller | Dice rolling animation app |
| GreetingCard | Birthday greeting card composable |
| HappyBirthday | Happy birthday image composable |
| Lemonade | Lemonade interactive state machine app |
| StudySnap | Study app with flashcards, quiz, note scanner (Compose Multiplatform) |
| SuperHeroes | Superheroes theme app |
| TipCalculator | Tip calculator UI |
| WoofApp | Dog breed showcase app with RecyclerView |

## Setup
1. Clone repo
2. Open in Android Studio
3. Use `./gradlew build` per project
4. `local.properties` excluded - set your SDK path

See `.gitignore` for exclusions. Ready to build!
